# Chrome Web Store listing copy

Paste these into the [Developer Dashboard](https://chrome.google.com/webstore/devconsole) when submitting.

## Extension name (suggested)

**Clipboard — Local Snippets**

(The manifest name is still `Clipboard`; you can use a longer display name in the dashboard if available.)

## Short description (≤ 132 characters)

Save text snippets locally. Search, categorize, copy, and export — no account, no cloud, no tracking.

## Detailed description

Clipboard is a fast, private snippet manager that lives entirely on your device.

**Local-first**
- Snippets and categories are saved with Chrome's local storage
- No login, no servers, no analytics
- Uninstalling removes extension data from your device

**Organize**
- Add text with Add / Search
- Create custom categories
- Filter by category or search across all snippets

**Work quickly**
- One-click copy from any row
- Edit, reorder, and delete with confirmation
- Optional animations and light/dark themes

**Your data, your control**
- Export a JSON backup anytime
- Import a previous backup
- Clear all data from Settings

**Permissions**
- `storage` — required to save snippets and preferences on your device only

**Support**
- https://github.com/shubhamgorule/clipboard/issues

## Permission justification (dashboard field)

> The storage permission saves the user's text snippets, category labels, and appearance preferences locally on the device using chrome.storage.local. No data is transmitted to external servers.

## Single purpose description

> Provides a local text snippet clipboard inside the browser toolbar popup.
