Privacy Policy — Clipboard (Chrome Extension)

Effective date: July 2, 2026

Clipboard is a local-first text snippet manager published by Shubham Gorule as an independent developer project. This document describes how the extension handles information and the terms under which you use it.

Summary

  • Your snippets, categories, and appearance preferences are stored on your device only.
  • We do not operate servers, accounts, analytics, or advertising in this extension.
  • Data does not leave your device unless you explicitly export a backup file.
  • You are responsible for backing up data you care about. The developer does not store or recover your snippets.

Information stored on your device

The extension may store the following in chrome.storage.local:

  • Text snippets you add or edit
  • Category labels you create
  • Appearance preferences (color mode, theme, animation setting)

This data is used solely to provide clipboard/snippet functionality inside the extension.

What we do not collect

  • No account registration or login
  • No cloud sync or remote backup by the developer
  • No sale, rental, or transfer of user data to third parties
  • No use of your data for advertising or creditworthiness

Permissions

The extension requests the storage permission so it can save your snippets and settings locally. It does not request broad host permissions or access to web page content.

When you copy a snippet, the extension writes text to your system clipboard using the browser clipboard API from the extension popup. It does not read your clipboard in the background.

Export, import, and deletion

  • Export — You can download a JSON backup from Settings.
  • Import — You can restore a previously exported JSON file from Settings.
  • Clear all data — You can delete all snippets, categories, and saved preferences from Settings.
  • Uninstall — Removing the extension from Chrome deletes locally stored extension data.

Because all data lives on your device, you are solely responsible for maintaining backups of any content you want to keep. We strongly recommend using Export regularly if your snippets are important.

Data loss and your responsibilities

Clipboard does not provide cloud backup, sync, or data recovery. You may lose stored snippets or settings if, for example:

  • You uninstall the extension or clear extension data in Chrome
  • You use Clear all data in Settings
  • Your browser profile is reset, corrupted, or removed
  • Your device fails, is lost, or is replaced without a backup
  • Browser or storage limits prevent saving new data
  • An import overwrites existing data

By using this extension, you acknowledge that keeping and protecting your data is your responsibility, not the developer’s.
